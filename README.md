# Agriculture-Résilience 2030 - Projet BI

## Équipe
- **Nou**: Data Engineering & ETL
- **Na**: Data Modeling & Visualization

## Stack Technique
- Talend Open Studio 8.0
- PostgreSQL 15
- Cube.js
- Metabase
- Docker & Docker Compose

## Installation
```bash
docker-compose up -d
```

## Structure
```
├── data/           # Données sources et traitées
├── etl/            # Jobs Talend
├── docker/         # Configurations Docker
├── sql/            # Scripts SQL
├── cube/           # Configuration Cube.js
└── docs/           # Documentation
```
